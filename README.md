# 幻兽帕鲁中文图鉴

一个独立的静态帕鲁图鉴项目，包含帕鲁总览、筛选、详情、属性图标、工作适应性图标、技能、掉落物与栖息区域互动地图。

## 本地运行

```bash
npm start
```

默认访问：

```text
http://127.0.0.1:4173/
```

项目是纯静态页面，也可以直接部署到 GitHub Pages、Vercel、Netlify 或 Cloudflare Pages。

## 页面

- `index.html`：总览页入口
- `pals.html`：帕鲁总览页
- `paldeck.html?pal=lamball`：帕鲁详情页

## 检查

```bash
npm run check
```
